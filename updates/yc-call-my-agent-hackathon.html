May 17, 2026 · With Arunachalam Barathidasan

Built Cosmic Browser Memory at the YC #CallMyAgent hackathon with Arunachalam Barathidasan.

Browser agents do not have a browsing problem. They have a memory problem.

Cosmic Browser Memory is a web traversal memory layer for browser agents. On each run it captures:

  • Page states and screenshots
  • Visual anchors on the page
  • Actions, coordinates, failures, and fixes
  • Successful workflows end to end
First run (cold): 1–2 minutes — repeated vision and planning calls After indexing (similar task): 20–40 seconds — often ~3x faster, one or zero vision calls Browser Use 2.0 (same task): 28–44 seconds — COSMIC's edge is compounding memory, not a single-run race

The bigger idea: Google indexed pages for humans. COSMIC indexes actions for agents.
